eShop

Zusammen mit meinen Kommilitonen Lars Obist und Yuliya Litvin einen elektronischen Onlineshop gebaut.
Hier kam dann Java als Programmiersprache zum Einsatz. Die GUI (graphical user interface) und die Datenbank sind von einander getrennt und mit einer Verwaltung kommunizieren sie miteinander. Damit mehrere Nutzer das Programm gleichzeitig verwenden können, wurden mit Sockets gearbeitet.

Gerne kannst du den eShop persönlich auf Github angucken und testen.
